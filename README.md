# Nama Aplikasi
Sistem Flask Katalog Baju

## Kegunaan Aplikasi
Aplikasi "Katalog Baju" ini adalah sebuah aplikasi web yang dirancang untuk membantu pengguna mencatat, mengelola, dan mengatur stok penjualanan baju mereka.
